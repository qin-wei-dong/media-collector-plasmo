1. 商店状态是否稳定，没有新的审核、权限或政策提醒。

本阶段只做观察和记录，不直接扩大功能范围。除非发现 P0/P1 阻塞，否则不急着发 hotfix。

## Hotfix 判定规则

满足任一条件时，优先考虑 v2.1.2 hotfix：

- 小红书主链路采集大面积失败。
- 扩展图标无法打开素材库。
- 已采集素材库无法加载或出现白屏。
- 导出主流程不可用，且失败重试无法恢复。
- Chrome Web Store 出现新的审核、权限或政策阻塞。

不满足以上条件时，先记录到候选池，等 24-48 小时观察结束后统一排期。
